# Java 学习梳理（基于“Java问题汇总”）

## 一、基础语法
- **标识符和命名规则**
  - 不能以数字开头，不能使用关键字。
  - 必须以字母、下划线或 `$` 开头。
  - 建议采用驼峰命名法。

- **数据类型与运算符**
  - 逻辑运算符：`&&`、`||`、`!` → 注意优先级。
    ```java
    System.out.println("bool1&&bool2:" + (bool1 && bool2));
    ```
  - 位运算符：`& | ^` → 与逻辑运算不同。

- **类与对象**
  - `new`、匿名对象。
  - `final` 类：不可被继承。
  - 抽象类与抽象方法：抽象类中可含抽象方法，子类必须实现。

---

## 二、字符串相关
- **String**
  - 不可变，每次修改都会生成新对象。
  - `getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin)` → 截取部分字符。

- **StringBuffer 与 StringBuilder**
  - 可变字符串，支持 `append`、`insert`、`setCharAt`。
  - `setCharAt` 会返回原字符。

- **常用方法**
  - `toCharArray()` → 字符串转数组。
  - `valueOf()` → 基本类型转字符串。
  - `getBytes()` vs `getChars()` → 前者处理字节，后者处理字符。

---

## 三、正则表达式与文本处理
- **Pattern 与 Matcher**
  - 用于模式匹配。
  - 示例：匹配网址
    ```java
    String regex = "(http://|www)\\.?\\w+\\.\\w+\\.\\w+\\.\\p{Alpha}+";
    Pattern p = Pattern.compile(regex);
    Matcher m = p.matcher("http://www.tsinghua.edu.cn");
    ```

- **常见用法**
  - 提取网址、邮箱、价格。
  - `replaceAll(regex, "")` → 删除匹配部分。

- **StringTokenizer**
  - 早期分割字符串方式：
    ```java
    StringTokenizer st = new StringTokenizer("苹果56.7元，香蕉12元");
    while(st.hasMoreTokens()) {
        System.out.println(st.nextToken());
    }
    ```
  - 现在推荐使用 `split()`。

---

## 四、输入输出与格式化
- **DecimalFormat**
  - 控制数字格式：
    ```java
    DecimalFormat df = new DecimalFormat("#,###.00");
    System.out.println(df.format(123456.789)); // 123,456.79
    ```

- **加密与解密**
  - 利用字符数组 + 密码数组循环加密。
  - 注意：乱码常由字符集编码不统一导致。

---

## 五、程序设计中的注意点
- **作用域**
  - 变量作用域不能重名。
  - `{}` 内声明的变量在外部不可见。

- **运算优先级**
  - 字符串拼接 `+` 的优先级低于逻辑运算符 → 需加括号。

- **异常处理**
  - `PatternSyntaxException`：正则语法错误。
  - 使用 `getDescription()` 获取错误信息。

---

## 总结
- **基础语法**：运算符、作用域、final、抽象类。
- **字符串处理**：String、StringBuffer、toCharArray、valueOf、getChars、getBytes。
- **正则与文本解析**：Pattern、Matcher、StringTokenizer、replaceAll。
- **输入输出与格式化**：DecimalFormat、加密解密。
- **错误调试**：异常处理、乱码原因分析。
